## Endpoints for entity
- GET     /    - all entities with pagination
- GET     /id  - one entity
- POST    /    - create
- PUT     /    - update
- DELETE  /id  - delete

## How to use
- backend
  - run in idea or ```mvn spring-boot:run```
- front
  - npm install
  - npm start

## Included
- backend
  - spring
    - security (auth + roles)
    - hibernate
    - boot
    - webflux
    - abstract controller, repository, entity for simple creating 
  - some tests
- front
  - [react](https://reactjs.org/)
  - [coreUi](https://coreui.io)
